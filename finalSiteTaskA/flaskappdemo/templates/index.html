Task A - Twitter Data Analysis

OVERVIEW

This task requires the analysis of data collected from Twitter from a text point of view.

It requires the collection of data via the Twitter API using hashtags relevant to the discussion around

Air Pollution in Delhi/NCR

and

Mumbai Rains.

The data collection can be done using hashtags that are trending on Twitter, such as #Smog, #MyRightToBreathe, #CropBurning, #MumbaiRains, #CycloneOckhi, etc. i.e. anything relevant to the topic. The analysis needs to be performed on the data collected. At lease 20,000 tweets (10,000 tweets for each issue) should be curated and stored in a MongoDB collection.
